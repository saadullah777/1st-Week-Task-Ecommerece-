****2nd Task Ecommerce****


This is a basic eCommerce backend application built with Node.js and EJS templating engine. It provides core functionalities required for an online store, including product listing, product details, and server-side

routing.

****Features****

Serve static pages (Home, Products, Product Details)

 RESTful routing system
 
 Backend server using Node.js & Express
 
 EJS templating engine for frontend rendering
 
 Simple and clean project structure

****Tech Stack****

──>  Node.js
 
──>  Express.js
 
──>  EJS
 
──> HTML, CSS

1. npm install
2. node server.js
3. Open http://localhost:3000
